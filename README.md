# Banking Transaction, Risk & Fraud Analysis System (MySQL)

This project simulates a real-world banking system designed to manage
customers, accounts, transactions, loans, cards, branches, employees,
and fraud detection analytics using MySQL.

It focuses on transaction monitoring, risk analysis, suspicious activity
detection, and customer financial insights.

## Technologies Used
- MySQL 8.0
- SQL (Advanced Level)

## SQL Concepts Covered
- Complex Database Design (10+ Tables)
- Primary & Foreign Keys
- Joins (INNER, LEFT)
- Aggregate Functions
- Subqueries
- CASE Statements
- Window Functions
- Views
- Date & Time Functions
- Fraud Detection Queries

## Business Use Case
- Banks & Financial Institutions
- Fraud Detection Systems
- Risk & Compliance Analysis
- Customer Financial Monitoring
